# Andersen_JavaCource

## Requirements

* JDK 11
* Apache Maven

## Build application:

```
mvn clean install
```

## Start application

To start an application:

```
 java -jar ./target/Andersen_JavaCource-1.0-SNAPSHOT.jar
```

Server up on [http://localhost:8090](http://localhost:8090).
